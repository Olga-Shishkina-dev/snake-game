# Snake Game
Моя первая игра на Python

Простая игра Змейка на Python с использованием Pygame. Собирайте еду, избегайте стен и себя!

## Как запустить
1. Установите Python и Pygame `pip install pygame`.
2. Скачайте репозиторий.
3. Запустите `main.py`.
4. Выберите сложность, прочитайте инструкции и играйте!

## Управление
- Стрелки движение.
- P пауза.
- ESC выход.

## Файлы
- `assets` звуки и картинки.
- `main.py` основной файл.
- `snake.py` класс змеи.
- `food.py` класс еды.
